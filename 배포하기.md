## 🧪 테스트 방법

### ✅ 회원가입 테스트 (초간단!)
1. https://proud-river-0710b0610.1.azurestaticapps.net/login.html 접속
2. "회원가입" 탭 클릭
3. **정보 입력:**
   - 이름: 아무거나 (예: 홍길동)
   - 이메일: 실제 이메일 (예: your@email.com)
   - 전화번호: 아무거나 (예: 123)
   - 비밀번호: **1자도 OK!** (예: 1, a, 아무거나)
   - 비밀번호 확인: 똑같이 입력
4. "회원가입하기" 클릭
5. 이메일 확인 (Supabase에서 자동 발송)
6. 확인 링크 클릭

### ✅ 로그인 테스트
1. 로그인 페이지로 자동 이동
2. 이메일/비밀번호 입력
3. "로그인하기" 클릭
4. 메인 페이지로 자동 이동
5. 우측 상단에 "👤 이름" 표시 확인 ✨

### ✅ 주문 테스트
1. 상품 클릭
2. "장바구니 담기"
3. 장바구니 페이지 이동 (우측 상단 🛒)
4. "💳 결제하기" 클릭
5. 배송 정보 입력
6. 주문 완료!

### ✅ 마이페이지 테스트
1. 우측 상단 "👤 이름" 클릭
2. 주문 내역 확인
3. 적립금 확인 (주문 금액의 3%)
4. 통계 확인 (총 주문, 총 구매액)
